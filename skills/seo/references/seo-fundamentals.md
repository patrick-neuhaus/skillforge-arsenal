# SEO Fundamentals — Conceitos Base

Fundamentos que todo profissional de SEO precisa entender antes de otimizar qualquer coisa.

---

## Como o Google Funciona

### 3 Fases
1. **Crawl** — Googlebot visita páginas seguindo links e sitemaps
2. **Index** — Google armazena e entende o conteúdo da página
3. **Rank** — Google ordena resultados por relevância e qualidade

### Implicações
- Se não é crawlável → não é indexado → não rankeia
- Se é indexado mas não é relevante → não rankeia
- Se é relevante mas não tem autoridade → perde pra concorrentes

---

## EEAT (Experience, Expertise, Authoritativeness, Trustworthiness)

Framework do Google pra avaliar qualidade de conteúdo. Não é fator de ranking direto, mas guia os quality raters humanos.

| Letra | O que é | Como demonstrar |
|:-----:|---------|----------------|
| E | Experience | Autor tem experiência real com o tema (reviews, casos, exemplos) |
| E | Expertise | Conhecimento técnico demonstrado (profundidade, precisão) |
| A | Authoritativeness | Reconhecido como autoridade (backlinks, menções, citações) |
| T | Trustworthiness | Site confiável (HTTPS, informações de contato, transparência) |

### YMYL (Your Money Your Life)
Nichos de saúde, finanças, jurídico — Google é EXTRA rigoroso com EEAT.

## Tipos de SEO

| Tipo | Foco | Exemplos |
|------|------|---------|
| Technical SEO | Infraestrutura do site | Speed, crawlability, structured data |
| On-Page SEO | Otimização individual por página | Title, meta, headings, conteúdo |
| Off-Page SEO | Sinais externos | Backlinks, menções, autoridade |
| Semantic SEO | Entidades e tópicos | Topical authority, NLP, schema |
| Programmatic SEO | Geração em escala | Templates + dados → páginas |
| Local SEO | Busca local | Google Business, NAP, reviews |

## Intenção de Busca

O fator MAIS importante pra rankear. Se o conteúdo não corresponde à intenção, não rankeia.

| Intenção | O que quer | Tipo de resultado | Exemplo |
|----------|-----------|:-----------------:|---------|
| Informacional | Aprender algo | Blog, guia, tutorial | "como funciona colchão de mola" |
| Investigativa | Comparar opções | Comparativo, review, lista | "melhor colchão de mola 2026" |
| Transacional | Comprar algo | E-commerce, landing page | "comprar colchão castor king" |
| Navegacional | Achar um site | Site oficial | "castor colchões site oficial" |

### Como identificar
1. **Pesquise no Google** — os resultados dizem a intenção
2. Se top 10 são blogs → informacional
3. Se top 10 são lojas → transacional
4. Se top 10 são comparativos → investigativa

## Fatores de Ranking (simplificado)

Os 200+ fatores do Google se resumem a:

| Categoria | Peso estimado | O que importa |
|-----------|:------------:|--------------|
| Relevância | ~30% | Conteúdo alinhado com intenção de busca |
| Autoridade | ~25% | Backlinks de sites relevantes e autoritários |
| Qualidade | ~20% | Conteúdo completo, preciso, atualizado (EEAT) |
| Experiência | ~15% | Core Web Vitals, mobile, HTTPS |
| Engajamento | ~10% | CTR, tempo na página, pogo-sticking |

**Nota:** Esses pesos são estimativas baseadas em estudos. Google não divulga pesos reais.

---

## SEO ≠ Hack

- SEO não é truque ou exploit — é alinhar conteúdo com o que pessoas buscam
- Google atualiza o algoritmo continuamente — táticas que "hackam" eventualmente quebram
- O que não muda: conteúdo de qualidade + relevância + experiência do usuário
- Foco no longo prazo: 6-12 meses pra resultados consistentes

---

## Ciclo de Trabalho SEO

```
1. Research: keyword research + análise de concorrentes
2. Plan: definir arquitetura + priorizar conteúdo
3. Create: produzir conteúdo otimizado
4. Optimize: on-page + technical + structured data
5. Promote: link building + distribuição
6. Measure: Search Console + Analytics
7. Iterate: ajustar baseado em dados
```

Repetir indefinidamente. SEO é processo contínuo, não projeto com fim.
