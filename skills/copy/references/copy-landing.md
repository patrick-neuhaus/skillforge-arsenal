# Copy — Landing Pages & Conversão

**Referências:** Joanna Wiebe / Copyhackers, Unbounce, CXL / Peep Laja

## Regra de Ouro
**Rule of One:** 1 oferta + 1 CTA + 1 público + 1 big idea.
Tudo que não serve a essa regra é distração — corta.

---

## 14 Tipos de Seção

Toda landing page de alta conversão é montada com seções desse menu:

| # | Seção | Objetivo | Quando usar |
|---|-------|----------|-------------|
| 1 | **Hero / Above the Fold** | Capturar atenção + CTA imediato | Sempre |
| 2 | **Social Proof Bar** | Logos, números, "como visto em" | Sempre que tiver |
| 3 | **Problem / Pain** | Nomear a dor com precisão (VoC) | Audiência nível 2-3 |
| 4 | **Solution / Benefits** | O que o produto faz e por que importa | Sempre |
| 5 | **How It Works** | 3-4 passos concretos | Produto com processo |
| 6 | **Testimonials** | Prova social específica (resultado + pessoa real) | Sempre |
| 7 | **Case Studies** | Prova de resultado com números | B2B, ticket alto |
| 8 | **Use Cases / Personas** | "Feito para quem..." | Produto com múltiplos perfis |
| 9 | **FAQ** | Objeções em formato de pergunta | Sempre (3-5 objeções) |
| 10 | **Comparison** | Vs. concorrente ou vs. sem-produto | Mercado competitivo |
| 11 | **Integrations** | Ecosistema compatível | SaaS, ferramentas |
| 12 | **Founder Story** | Humaniza a marca, origem | Marca pessoal, premium |
| 13 | **Demo / Tour** | Mostra o produto em ação | SaaS, software |
| 14 | **Guarantee** | Remove risco da compra | Antes do CTA final |

## CTA Formula

```
[Action Verb] + [What They Get] + [Qualifier opcional]
```

| Fraco | Forte |
|-------|-------|
| Submit | — |
| Sign Up | Criar minha conta grátis |
| Learn More | Ver como funciona |
| Buy Now | Garantir meu acesso |
| Get Started | Começar meu teste de 14 dias |
| Download | Baixar o template gratuito |
| Register | Reservar minha vaga |

**Regras de CTA:**
- Verbo de ação no imperativo ou primeira pessoa
- Reforce o que o usuário GANHA, não o que ele FAZ
- Repita o CTA pelo menos 3x na página (acima do fold, meio, final)
- Micro-copy abaixo do botão remove risco: "Sem cartão de crédito", "Cancele a qualquer momento"

## 5 Templates de Landing Page

### Template 1 — Compact (tráfego pago, alta intenção)
Melhor para: Google Ads, audiência quente, produto simples
Seções: Hero + Social Proof + Benefits + How It Works + CTA
Comprimento: curto (~600 palavras)

### Template 2 — Varied Engaging (cold traffic, B2C)
Melhor para: Facebook/Instagram, audiência fria, produto que precisa de contexto
Seções: Hero + Problem + Solution + How It Works + Testimonials + FAQ + Guarantee + CTA
Comprimento: médio (~1200 palavras)

### Template 3 — Enterprise / B2B
Melhor para: SaaS B2B, ticket médio-alto, múltiplos decisores
Seções: Hero (ROI-focused) + Social Proof Bar + Problem + Solution + How It Works + Case Studies + Integrations + FAQ + Guarantee + CTA
Comprimento: longo (~1800 palavras)

### Template 4 — Product Launch
Melhor para: lançamento, urgência, pré-venda
Seções: Hero + Backstory + Problem + Solution + Testimonials + Bonus Stack + Pricing + Urgency/Countdown + Guarantee + FAQ + CTA
Comprimento: longo (~2000+ palavras)

### Template 5 — Feature-Heavy (anti-padrão — evite)
Listas de features sem narrativa = baixa conversão. Use apenas para audiência já convencida (Most Aware).

## Princípios de Copy para Landing Page

1. **Headline consome 80% do investimento** (Ogilvy) — escreva 20 versões, escolha a melhor
2. **Pessoas escaneiam, não leem** — otimize para H1/H2, bullets, bold, números
3. **Message Match** — headline da LP deve corresponder exatamente ao anúncio/fonte de tráfego
4. **Comprimento depende do nível de consciência** — Unaware = longo | Most Aware = curto
5. **Above the fold deve ter:** headline claro + sub-headline + CTA + social proof mínimo
6. **Objeções no FAQ** — não espere o usuário ter dúvidas; antecipe as 5 principais
7. **Prova específica > prova genérica** — "2.847 clientes" > "milhares de clientes"

## CRO Analysis Framework — 7 Dimensões

Quando auditar uma landing page, avalie cada dimensão com nota 1-5:

| Dimensão | O que avaliar |
|----------|--------------|
| **1. Clareza** | A proposta de valor fica clara em 5 segundos? |
| **2. Relevância** | O copy fala com a audiência certa no contexto certo? |
| **3. Proposta de Valor** | A diferenciação é clara vs. alternativas? |
| **4. Fricção** | O formulário/checkout tem steps desnecessários? |
| **5. Distração** | Há elementos que desviam do CTA principal? |
| **6. Confiança** | Tem prova social, garantia, credenciais visíveis? |
| **7. Urgência** | Há motivação para agir agora vs. depois? |

**Diagnóstico rápido:**
- Score abaixo de 3 em Clareza ou Proposta de Valor = reescreva antes de testar o resto
- Score abaixo de 3 em Confiança + sem testemunho específico = conversão vai travar no meio do funil
- Score abaixo de 3 em Urgência = alta taxa de "vou pensar"
