# Copy Process

## Workflow Completo (7 Etapas)

```
1. Research/Briefing
   └── produto, mercado, audiência, concorrentes, diferenciação

2. VoC Mining
   └── linguagem exata do cliente (não interpretação sua)

3. Copy Brief
   └── define escopo, audiência, tom, objetivo, restrições

4. Outline
   └── qual framework, quais seções, ordem dos argumentos

5. First Draft
   └── escreva sem editar — autocrítica prematura mata criatividade (Halbert)

6. Edição — Seven Sweeps
   └── 7 passagens sequenciais com foco exclusivo cada

7. Teste
   └── A/B testing, validação com dados, iteração
```

## VoC Mining — Voice of Customer

**Por que:** A linguagem do cliente converte mais que a sua. "Nosso analytics com IA traz insights" é corporativo. "Eu finalmente consegui ver onde estava perdendo dinheiro" é VoC.

**Onde encontrar:**

| Fonte | O que capturar |
|-------|---------------|
| Amazon reviews ⭐⭐⭐⭐⭐ | Aspiração, resultado desejado |
| Amazon reviews ⭐ | Dor, frustração, expectativa não atendida |
| Reddit (r/[nicho]) | Linguagem informal e honesta, gírias do segmento |
| Tickets de suporte | Problemas reais, palavras exatas do cliente |
| Pesquisas e entrevistas | Contexto profundo, motivação real |
| Quora, Indie Hackers, HN | Dúvidas específicas, objeções comuns |
| YouTube comments | Reações emocionais, o que ressoa |
| G2 / Capterra | Comparações, switching triggers |

**Review Mining (método CopyHackers):**
1. Abra Amazon/G2/Capterra na sua categoria
2. Leia reviews de 5 estrelas e 1 estrela
3. Destaque frases que descrevem: dor antes, resultado depois, surpresa, frustração
4. Use essas frases EXATAS no seu copy — não parafraseie

## Copy Brief — Template Completo

```markdown
## Copy Brief

**Objetivo:** [conversão / awareness / nurture / reativação]

**Audiência:**
- Persona: [quem é essa pessoa]
- Nível de consciência (Schwartz): [1-5]
- Maior dor/frustração: [frase do cliente, não sua]
- Maior desejo: [resultado que quer]

**Proposta de valor:** [1 frase que diferencia]

**Tom de voz:** [3 adjetivos: ex. direto, humano, confiante]

**Prova disponível:**
- Testemunhos: [sim/não — específicos?]
- Números: [ex: 2.847 clientes, 94% satisfação]
- Cases: [sim/não]

**Objeções principais a endereçar:**
1.
2.
3.

**CTA desejado:** [ação específica]

**Restrições:**
- Tamanho: [limite de caracteres ou palavras]
- Plataforma: [onde vai ao ar]
- Compliance: [restrições jurídicas/regulatórias]
```

## Seven Sweeps — Framework de Edição

Sete passagens sequenciais sobre o copy. Cada passagem tem foco exclusivo — não tente fazer tudo de uma vez.

| # | Sweep | Foco | Pergunta-chave |
|---|-------|------|----------------|
| 1 | **Clarity** | O leitor entende? | Está claro sem esforço cognitivo? |
| 2 | **Voice & Tone** | Soa consistente? | Tom combina com a marca e o canal? |
| 3 | **So What** | Por que importa? | Cada afirmação tem benefício explícito? |
| 4 | **Prove It** | Tem evidência? | Claims têm prova concreta? |
| 5 | **Specificity** | É concreto? | Números, prazos, detalhes específicos? |
| 6 | **Heightened Emotion** | Faz sentir? | Tem transformação emocional ao longo? |
| 7 | **Zero Risk** | Removeu barreiras? | Garantia + trust elements perto do CTA? |

### Sweep 3 — Teste "So What"
Após cada afirmação, pergunte: "E daí? O que o leitor ganha com isso?"

| Fraco | Forte |
|-------|-------|
| "Nossa plataforma usa analytics com IA" | "Analytics que traz insights que você perderia manualmente — decisões melhores em metade do tempo" |
| "Integramos com mais de 200 ferramentas" | "Funciona com o que você já usa — zero migração, zero treinamento" |

### Sweep 5 — Upgrades de Especificidade

| Vago | Específico |
|------|------------|
| Save time | Save 4 hours every week |
| Many customers | 2,847 teams |
| Fast results | Results in 14 days |
| Great support | Response within 2 hours |
| Affordable | From R$47/month |
| Easy to use | Up and running in 15 minutes |

### Sweep 7 — Risk Reducers (perto do CTA)
- Money-back guarantee (período específico)
- Free trial (sem cartão de crédito)
- "Cancel anytime"
- Prova social específica (número + categoria de cliente)
- Explicação clara do próximo passo ("Você receberá X em Y minutos")

### Sweep 6 — Heightened Emotion (Detalhamento)

Se o copy está "sem vida" ou lê como manual técnico, aplique:

**1. Linguagem Sensorial**
- Substitua abstrações por experiências: "melhorar produtividade" → "finalmente sair do escritório às 18h"
- Use verbos sensoriais: ver, sentir, ouvir, tocar, perceber
- Descreva o MOMENTO da transformação, não o resultado abstrato

**2. Antes/Depois Emocional**
- Antes: "Você fica até tarde, ansioso, checando WhatsApp a cada 5 minutos"
- Depois: "Você dorme tranquilo sabendo que cada cliente foi respondido"
- A distância emocional entre antes e depois = intensidade do desejo

**3. Life Force 8 (Ca$hvertising)**
Conecte ao desejo biológico primário mais relevante:
- Sobrevivência / extensão de vida
- Liberdade do medo e da dor
- Condições de vida confortáveis
- Ser superior / vencer / não ficar pra trás
- Cuidar e proteger entes queridos
- Aprovação social / status

**4. Pratfall Effect**
- Mostre vulnerabilidade estratégica: "Antes de descobrir isso, eu perdia 3 clientes por semana"
- Competência + falha = mais simpático e confiável

**5. Peak-End Rule**
- O clímax emocional E o final do copy são os mais lembrados
- CTA final deve ser o momento de maior emoção/motivação

## Word-Level Quick Checks (Edição Final)

**Cortar sem dó:**
- very, really, just, actually, basically, "in order to", "the fact that"

**Substituir:**
| Corporativo | Humano |
|-------------|--------|
| utilize | use |
| implement | set up |
| leverage | use |
| facilitate | help |
| seamless | smooth |
| robust | forte/confiável |
| synergy | (delete) |

**Vigiar:**
- Advérbios (-mente excessivos)
- Voz passiva → ativa
- Nominalizações: "make a decision" → "decide", "give consideration to" → "consider"

## 8 Copy Problems & Fixes

| Problema | Sintoma | Fix |
|---------|---------|-----|
| **Wall of Features** | Lista de features sem benefício | Adicione "which means..." após cada feature |
| **Corporate Speak** | Tom frio, distante, palavras longas | "Como um humano diria isso?" |
| **Weak Opening** | Abertura genérica ou sobre a empresa | Comece com problema ou resultado desejado |
| **Buried CTA** | CTA escondido no final, uma vez só | CTA óbvio, cedo e repetido ao longo |
| **No Proof** | Claims sem evidência | Testimonials específicos, números, casos |
| **Generic Claims** | "Somos os melhores", "qualidade superior" | Especificar quem, como e quanto |
| **Mixed Audiences** | Fala com todo mundo = fala com ninguém | Escolher UMA audiência |
| **Feature Overload** | 10+ benefícios → paralisia | 3-5 benefícios-chave, os mais relevantes |
