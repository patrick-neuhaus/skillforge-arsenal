# Power Words & Tom de Voz

## Power Words (por categoria)

### Urgência
agora, hoje, imediato, urgente, limitado, última chance, restam [N], expira, deadline, encerra, somente até, não perca

### Exclusividade
exclusivo, secreto, VIP, só para membros, acesso antecipado, bastidores, seleto, reservado, especial, por convite

### Confiança
comprovado, garantido, certificado, testado, aprovado, recomendado, seguro, verificado, auditado, reconhecido, premiado

### Curiosidade
surpreendente, revelado, pouco conhecido, escondido, a verdade sobre, o que poucos sabem, descoberto, finalmente revelado

### Valor
grátis, bônus, economize, incluso, desbloqueie, transforme, conquiste, ganhe, maximize, multiplique, eleve

### Emocional
finalmente, imagine, merece, liberdade, paz, segurança, orgulho, confiança, alívio, satisfação, vitória, independência

### Resultado
resultado, conquista, transformação, melhoria, aumento, redução, dobrou, triplicou, economizou, eliminou

### Facilidade
simples, fácil, rápido, sem complicação, passo a passo, automático, pronto, em minutos, sem esforço

## Palavras a EVITAR

### Vazias (cortam sem perda nenhuma)
- muito, realmente, bastante, extremamente
- apenas, simplesmente, basicamente, na verdade
- de certa forma, por assim dizer, de algum modo

### Corporativas (substitua pelo humano)
| Corporativo | Substitua por |
|-------------|--------------|
| utilize | use |
| implementar | configurar, colocar em prática |
| alavancar | usar, aproveitar |
| facilitar | ajudar |
| otimizar | melhorar |
| sinergia | (delete) |
| robusto | forte, confiável |
| seamless | sem atrito, suave |
| holístico | completo |
| proativo | (contexto) antecipado, preventivo |

### Hedging (criam incerteza)
- talvez, possivelmente, poderia, deveria considerar
- tente, provavelmente, aparentemente
- "pode ser que", "acredito que"

### Voz Passiva → Ativa
| Passiva | Ativa |
|---------|-------|
| "foi decidido" | "decidimos" |
| "é recomendado" | "recomendamos" |
| "pode ser feito" | "você faz" |
| "será entregue" | "entregamos" |

### Clichês desgastados
- "transformador", "disruptivo", "inovador" (sem prova)
- "líderes do mercado"
- "solução completa"
- "one-stop-shop"
- "best-in-class"

## Tom de Voz Framework

### Como definir tom de voz

1. **Defina 3-4 personality traits** — ex: ousado, direto, empático, técnico
2. **Para cada trait, liste DO e DON'T**
3. **Adapte por canal** (ver tabela abaixo)
4. **Teste de voz:** "Se a marca fosse uma pessoa, como falaria nessa situação?"

### Exemplo de trait documentado

**Trait: Direto**
- DO: Frases curtas. Vai direto ao ponto. Não enrola antes de fazer a afirmação principal.
- DON'T: Não abre com "Olá! Espero que você esteja bem." Não usa 3 parágrafos antes do ponto.

**Trait: Humano**
- DO: Contração de palavras (você tá, a gente), imperfeições calculadas, histórias pessoais.
- DON'T: Nunca soa como comunicado corporativo. Nunca usa "prezado(a)".

### Adaptação por Canal

| Canal | Tom | Ajuste |
|-------|-----|--------|
| WhatsApp | Conversacional, informal | Como se fosse uma mensagem de amigo |
| Email marketing | Semi-formal, pessoal | Tom de carta, não broadcast |
| LinkedIn | Profissional, mas humano | Insights com perspectiva pessoal |
| Instagram | Visual, emocional | Curto, impactante, sensorial |
| Landing page | Direto, orientado a resultado | Benefícios acima do fold, sem rodeios |
| Blog/SEO | Educacional, útil | Didático, sem jargão desnecessário |
| Ads | Disruptivo, específico | Hook forte, benefício imediato |

## Checklist Final de Linguagem

Antes de publicar:
- [ ] Cortei todos os advérbios desnecessários?
- [ ] Substitui linguagem corporativa por linguagem humana?
- [ ] Voz ativa ao longo de todo o texto?
- [ ] Frases em média abaixo de 20 palavras?
- [ ] Cada parágrafo tem no máximo 3 linhas?
- [ ] Tom é consistente do início ao CTA?
- [ ] Usei pelo menos 1 power word de urgência ou valor?
